# web-scraping-2022

Урок по scrapy.
Задание:

1) Создать пауков по сбору данных о книгах с сайтов labirint.ru и/или book24.ru
2) Каждый паук должен собирать:
  * Ссылку на книгу
  * Наименование книги
  * Автор(ы)
  * Основную цену
  * Цену со скидкой
  * Рейтинг книги
3) Собранная информация должна складываться в базу данных
